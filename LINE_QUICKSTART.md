# 🚀 เริ่มใช้งาน LINE Notify (5 นาที)

## ขั้นตอนที่ 1: สร้าง LINE Token

1. เปิด: **https://notify-bot.line.me/**
2. กด **"เข้าสู่ระบบ"**
3. ล็อกอินด้วยไลน์
4. กด **"My page"** (มุมบนขวา)
5. เลื่อนลงล่าง กด **"Generate token"**
6. กรอก:
   - Token name: **"บัญชีรายจ่ายสวน"**
   - Select: **"1-on-1 chat with LINE Notify"**
7. กด **"Generate token"**
8. **คัดลอก Token** (จะแสดงครั้งเดียว!)

---

## ขั้นตอนที่ 2: ใส่ Token

1. เปิดไฟล์ **`backend/.env`**
2. หาบรรทัด:
   ```
   LINE_NOTIFY_TOKEN=
   ```
3. วาง Token:
   ```
   LINE_NOTIFY_TOKEN=tPZ8xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
   ```
4. **บันทึกไฟล์**

---

## ขั้นตอนที่ 3: Restart Backend

```bash
# หยุด Backend (กด Ctrl+C)
# รันใหม่
cd backend
npm start
```

ควรเห็น:
```
💬 LINE Notify: ✅ Configured
```

---

## ขั้นตอนที่ 4: ทดสอบ

### วิธีที่ 1: เพิ่มรายจ่าย
1. เปิด http://localhost:3000
2. คลิก "เพิ่มรายจ่าย"
3. กรอกข้อมูล และบันทึก
4. **เช็คไลน์ → จะได้รับแจ้งเตือน!**

### วิธีที่ 2: กดปุ่มส่งสรุป
1. คลิกปุ่ม **"💬 ส่งสรุปไปไลน์"**
2. เช็คไลน์ → จะได้รับสรุปรายจ่ายวันนี้
